IComparer

IEnumerable : foreach문을 쓰기위한 인터페이스
	     foreach 문의 in 뒤에 배열명이 와야 쓸수있는데 다른거도 넣어서 쓰기위해 쓰는 인터페이스
	     자주 사용하지는 않음

구조체 : 값타입 (value)
	인스턴트 생성을 new로해도 되고 안해도됨
	기본생성자는 명시적으로 정의할 수 없다

readonly 와 const의 차이 : const는 컴파일할때부터 값이 정확해야함 이게 상수
		          readonly는 컴파일 타임에 값이 없어도 된다 생성할때 값을 넣어줘도 되지만 한번 생성하면 값을 바꿀수가 없다

델리게이트 : 미리 함수를 다 등록을 해놓고 당장 메서드를 호출하지 않고 원할때 호출하고 싶을때 사용 => 델리게이트(위임)
	    델리게이트는 객체를 통해 원할때 메서드 호출
	    델리게이트는 연산이 가능 !!
	    델리게이트는 사용자 정의 타입임!!

이벤트 : 델리게이트가 있어야 이벤트 정의가능 
	이벤트 정의 문법 : event 델리게이트명 이벤트명
